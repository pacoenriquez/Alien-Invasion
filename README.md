# Alien-Invasion
Alien Invasion, from the book "Python Crash Course", is a game project  to learn and practice Python.
## Requirements
* Python3
* Pygame
## Instructions
* Left and right Arrows to move the ship
* Space to fire bullet
* "q" to exit the game
